# Minions Contracts

Agreements, statements of work, terms, and signed documents

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-contracts/sdk`) — Core library for Node.js
- **Python SDK** (`minions-contracts`) — Python bindings
- **CLI Tool** (`@minions-contracts/cli`) — Command-line interface
- **Web App** — Interactive playground at [contracts.minions.wtf](https://contracts.minions.wtf)
