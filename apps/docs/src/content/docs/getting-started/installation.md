---
title: Installation
description: Install Minions Contracts packages
---

## TypeScript / Node.js

```bash
npm install @minions-contracts/sdk minions-sdk
```

## Python

```bash
pip install minions-contracts
```

## CLI

```bash
npm install -g @minions-contracts/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
